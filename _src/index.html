Projects

Following are a few of my projects, in no particular order.

  • New Heroku Dashboard

    Currently the major project I'm working on at Heroku. An ambitious rewrite of the Heroku Dashboard, it's an Ember.js app backed by a set of small, sharp Node modules made for interacting with the Heroku API and other parts of the platform.

  • RestModel

    A library for working with APIs in Ember apps, using localStorage as a cache.

  • heroku-proxy

    A piece of Express middleware for proxying requests to the Heroku Platform API.

  • node-heroku-client

    A Node module for making requests to the Heroku Platform API. It is generated automatically from the API schema.

  • worker-monitor

    A Node module for monitoring workers and gracefully shutting down and restarting them.

  • steeltoe

    A JavaScript module for traversing objects which supports autovivification.

  • clusterflock

    A clustering HTTP server for Node apps.

  • Detour

    A Rails engine for managing feature rollouts.

  • so_stub_very_test

    Simplified stubs for testing an Excon API client.
